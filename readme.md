Based on LPC8N04FHI33 with RTC calendar, temperature sensor, OLED panel, LEDs  
  
Low power @8uA total board  
Support battery less mode  
Support temperature log  
  
Support Android APK, phone compatible with  
Nexus6  
Huawei Mate10  
etc
